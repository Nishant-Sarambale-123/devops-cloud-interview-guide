## **3Ô∏è‚É£ Import Existing Resources into Terraform State**

Use the import command:

```bash

```

Terraform now knows:

* This resource exists
* The state is linked to Terraform

Import other resources similarly:

```bash

terraform import aws_security_group.app sg-123456
terraform import aws_vpc.main vpc-abc123
```

* **No changes** ‚Üí your code matches exactly
* **Small differences** ‚Üí update your TF code to reflect reality
