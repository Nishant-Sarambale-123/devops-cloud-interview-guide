# **75. Write Terraform code to create any resource on AWS**

---

## **1. Question**

**Write Terraform code to create any AWS resource.**

---

## **2. Short explanation**

Terraform requires a provider block and a resource block. A minimal configuration can create an AWS resource like an EC2 instance, S3 bucket, or VPC.

---

## **3. Answer**

Below is a simple Terraform example that creates an **S3 bucket** on AWS using AWS provider and one resource block.

---

## **4. Detailed explanation**

Here is the full, clean, production-ready example.

### **A) Provider Configuration**

You must specify which cloud provider Terraform should use.

```hcl
terraform {
  required_providers {
    aws = {
      source  = "hashicorp/aws"
      version = "~> 5.0"
    }
  }
}

provider "aws" {
  region = "ap-south-1"
}
```

---

### **B) Create an S3 Bucket**

```hcl
resource "aws_s3_bucket" "my_bucket" {
  bucket = "nishant-demo-bucket-12345"
  tags = {
    Environment = "dev"
    Owner       = "Nishant"
  }
}
```

This code will:

* Initialize AWS provider
* Create an S3 bucket
* Add tags

---

### **Alternative Example: EC2 Instance**

If the interviewer wants an EC2 example, here is a minimal version:

```hcl
resource "aws_instance" "web" {
  ami           = "ami-0e670eb768a5fc3d4" # Amazon Linux 2 (Mumbai)
  instance_type = "t2.micro"

  tags = {
    Name = "demo-server"
  }
}
```

---

### **Steps to apply**

Run:

```
terraform init
terraform plan
terraform apply
```
